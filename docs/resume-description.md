# 简历项目描述

## 精简版

**绿智顾问——企业绿色转型与出海合规AI工作台**  
基于 Vue 3、FastAPI、Dify、N8N、MySQL、Redis 和 Docker 构建企业材料解析、画像抽取、绿色成熟度诊断、RAG风险引用及报告生成闭环；使用确定性规则实现可解释评分，支持字段证据、跨文件冲突隔离、低依据人工审核和PDF导出；实现Dify失败本地降级、Redis故障MySQL回退、运行质量监控及30条场景评测。

## 可量化版

- 设计15项企业绿色画像字段及5维100分评分规则，实现字段—证据—规则—分数全链路追溯；
- 建立4类官方知识来源的本地RAG基线，风险输出包含来源、摘要和匹配分，无依据结论自动转人工；
- 实现Dify输出证据校验、资料冲突检测、Redis/MySQL双层任务状态及Docker健康检查；
- 编写22项自动化测试和30条场景评测，覆盖抽取、检索、缺失、冲突、报告与故障降级。

不要把“30/30固定评测通过”写成“准确率100%”。面试时应明确它是当前合成基线。
